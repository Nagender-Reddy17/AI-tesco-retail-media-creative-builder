# AI-Powered Tesco Retail Media Creative Builder

## Overview
The AI-Powered Tesco Retail Media Creative Builder is a prototype tool designed to help advertisers quickly generate Tesco-compliant retail media ad creatives using artificial intelligence.

The tool simplifies the creative production process by combining AI-based text generation with a Tesco guideline compliance rules engine, enabling faster turnaround and reducing the risk of non-compliant ads.



## Problem Statement
Creating retail media creatives is often time-consuming, costly, and dependent on agencies. Many creatives fail compliance checks, leading to campaign delays and increased operational overhead.

There is a need for a simple, automated solution that allows advertisers to generate professional, Tesco-compliant creatives instantly without design expertise.



## Solution
This project provides:
- An AI-powered creative generator
- A rules-based Tesco compliance validation engine
- A clean, easy-to-use web interface
- Instant creative preview with compliance status



## Key Features
- AI-generated retail ad copy
- Tesco guideline compliance validation
- Clean and intuitive UI
- Fast creative generation (seconds)
- Scalable architecture for future enhancements
